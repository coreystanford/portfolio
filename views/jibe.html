This project is setup to allow access to the inner pages only once an individual has signed up and logged into the system. The only page that is available to anonymous users is the homepage of the site.

The site is designed to make it easy to upload and share creative projects, and gain inspiration from other projects shared within the network. There are two main feeds - the personal feed, and the explore feed. The explore feed showcases the most recent additions to the site, which users can scroll through via an infinite loader. The personal feed shows the user only the projects from profiles they have chosen to follow within the network.

There is a lot of functionality that is built into the network, such as a full reporting system for inappropriate content, a customizable slider on the profile page, and a top-tier content management system (which can delete users, edit categories, edit the homepage, and even control the number of projects loaded in the feeds).

Key Challenge:

The most challenging segment of this project, besides keeping the files organized as the project scaled, was setting up the modal window within the feeds. This window allows the user to seemlessly view the content in the feed without losing their scroll position. The system uses jQuery to pass an ID in the feed (placed by PHP) to an AJAX request that passes the ID to a database request (based on the ID), and dynamically returns the results into the modal template. Implementing this feature required a lot moving parts, but it is well worth the effort because of its smooth end-user experience.
