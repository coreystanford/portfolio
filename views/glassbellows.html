Visit Site

Primary Goal:

Make a fluidly responsive website that showcases imagery well.

Solo Project

Technologies:

  • HTML
  • CSS
  • JavaScript

Software:

  • Sublime Text 3
  • Photoshop CS6
  • Illustrator CS6

Timeframe:

10 days

In the winter of 2013 I was looking for a unique challenge in my spare time, and so I took up the 19th century artform of wet plate collodion. This process is chemically-intensive, time-consuming, and very complicated. I wanted to create a modern, responsive site that would nicely showcase the photography, as well as clearly explain the process in a simple, straightforward manner.

The logo is inspired by the 8Ã—10 Deardorff V8 View Camera from 1952 I use to make the tintypes.

The main challenge in developing this template arose when attempting to get the CSS positioning of horizontal colour blocks to align nicely at the outer margins of the template. The mobile menu was also developed completely with CSS, and utilizes a checkbox to set the status of the menu.

NOTE: The 'home' page and 'about' page are the only HTML pages that were developed.
